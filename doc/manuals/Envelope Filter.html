Envelope Filter

Functionality

Envelope Filter is a Filter whose frequency is set according to the envelope of the input or sidechain signal. It is also known as "Auto Wah" effect. The behavior of the envelope can be altered by Attack, Release and Response settings.

Tips

Imagine you already have adjusted the Activation knob and the filter is wobbling nicely between Lower and Upper frequency but you would still like to alter the amount of time it remains in the upper or lower frequency range use the Response setting to alter the dynamics of your signal:

  • If the filter opens less than expected raise the Response value
  • If the filter is open most of the time reduce the Response value

If you want Envelope Filter to operate reversed just switch upper and lower handles in the frequency response graph.

Controls

  • Input Gain: Control the input level before the signal is processed
  • Input Level: The level after the input knob
  • Clip: Flashes if input level rises above 0dB
  • Bypass: Don't process anything, just bypass the signal
  • Sidechain: Use the sidechain instead of the original signal input to determine the envelope
  • Output: Control the output of the plugin - clipping could destroy your signal
  • Output Level: The level after processing and after the output knob
  • Clip: Flashes, if the signal rises about 0dB in the output stage
  • Mix: Determine the amount of the filtered signal in the output.
  • Output Gain: The level at the output of the processor
  • Activation: Alter the gain of the input or sidechain signal before the envelope is calculated
  • Attack: The amount of time the envelope needs to follow the attack phase of the signal
  • Release: Amount of time the envelope needs to fall back to its minimum
  • Response: Alter the dynamics of the input signal. Use negative values to enhance (like gating would do) or positive values to reduce (like a compressor would do) the dynamics of the signal.
  • Resonance: Add an accentuation to the center frequency. Used on a bandpass the surrounding frequencies are decreased in level and the band gets more narrow, a bandreject is narrowed down to a thin needle
  • Mode: Choose the type of the filter
  • Lower: The lowest frequency of the filter
  • Upper: The highest frequency the filter could be set to
